# Virtusa-CorePython-FareCalc-Travel-Optimizer
FareCalc is a simple Python-based console application that calculates ride fares for a ride-sharing service called CityCab.
It determines the final fare based on:
--> Distance traveled
--> Vehicle type
--> Time of the ride (for surge pricing)

# How to Run
Step 1: Open Terminal / VS Code Terminal
Step 2: Navigate to project folder
cd "D:\python project"
Step 3: Run the program
python farecalc.py
